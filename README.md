# MiniPhotoShop

**Projenin fonksiyonları**
1. Dosyadan Fotoğraf alma
2. Renk Değiştirme
   - Negatif
   - Gri Tonlama
 3. Döndürme ve Aynalama
    - Sağa Çevirme
    - Sola Çevirme
    - Dikey Aynalama
    - Yatay Aynalama
 4. Ölçeklendirme
 5. RGB Kanallarını gösterme
    - Red
    - Green
    - Blue
 6. Fotoğrafın üstündeki imlecin RGB değerleri ve renk gösterimi
 7. Histogram Çizdirme
    - Renkli Histogram
    - Gri histogram
 8) Resmi Kaydetme
